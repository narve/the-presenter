Live Markdown Presenter

The presentation will launch in a new tab, which can be kept in sync with your presentation, if you enable "Live update presentation".

If you enable "Save content to local storage", your Markdown content will be saved in your browser's local storage, so it will persist across page reloads.

When you start the presentation, a unique URL will be generated for it in the new tab. This link can also be used to share the presentation with others.
